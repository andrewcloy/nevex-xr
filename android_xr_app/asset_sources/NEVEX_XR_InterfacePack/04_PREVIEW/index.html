Key Direction

Use the pack as a controlled design system, not as a floating dashboard of equal icons. Labels are required in menu navigation, and modes should change emphasis without creating a second menu tree.

HUD / Live View

Prefer glyph icons, sparse layout, edge-safe placement, and minimal persistent clutter.

Menus / Panels

Prefer tile icons for browseable settings, profile entry points, and quick control surfaces.
