# Screen Inventory

- `live_hud_minimal`: scene-first view with mode, battery, link, capture, and optional recenter
- `live_hud_expanded`: temporary richer HUD for active sensor tuning or mission context
- `no_hud_clean_view`: pure scene view with only emergency overlays
- `quick_controls_strip`: fast live adjustments for brightness, zoom, IR, fusion, and detection
- `sensor_fusion_panel`: blend strength, overlay mode, edge enhancement, and fallback logic
- `thermal_panel`: palette, blend, hotspot bias, and thermal alignment entry
- `low_light_panel`: gain, exposure, brightness, focus assist, and IR control
- `detection_panel`: person/animal detection, target lock behavior, and highlight intensity
- `record_playback_panel`: record, snapshot, playback, export, storage view
- `device_status_panel`: battery, charging, Wi-Fi, Jetson, storage, temperature, FPS, latency
- `profile_select_panel`: tactical, hunter, police_patrol, search_and_rescue, recreational
- `connectivity_panel`: link pairing, reconnect status, Jetson handshake, offline state
- `calibration_panel`: stereo align, thermal align, recenter, guided checks
- `diagnostics_panel`: engineering metrics, sensor states, validation snapshots, debug actions
- `startup_connect_panel`: initial device discovery, pairing, and safe recovery path
- `alert_cards`: link lost, low battery, recording active, sensor fault, overheat, critical fault
